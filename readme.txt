Tested up to: 3.4.2
Stable tag: 2.2

Adds planwise iframe via a shortcode or via a page template

== Description ==
using the shortcode of *[planwise]* or copying the planwise template into your theme and creating a page with it, you can show the planwise iframe easily on your website.
